# Моблильное приложение "Мессенджер"

---

## Описание проекта

Мобильное приложение "Мессенджер" — это отечественное приложение для обмена сообщениями и медиафайлами, 

**Цель проекта:**
- Создание приложения для обмена сообщений
- Разработка аналога не отечественным сервисам для обмена сообщениями
---

## Основные возможности

- **Регистрация и авторизация** – создание учетной записи по номеру телефона
- **Управление профилем** – изменение настроек и данных аккаунта
- **Обмен сообщениями** – отправка текстовых, голосовых сообщений другим пользователям
- **Файлообмен** – отправка фото, видео и документов разных форматов другим пользователям
- **Групповые чаты** – создание и управление группаовыми чатами
- **Статус пользователя** – отображение статуса онлайн или офлайн
- **Уведомления** – оповещение пользователя о новых сообщениях

## Технологический стек

### Backend
- Firebase Cloud Firestore
- Firebase Realtime Database
- Cloud Storage for Firebase
- Firebase Authentication
- Firebase Cloud Messaging
- Spring Boot

### Frontend (Android)
- Kotlin
- Jetpack Compose
- ViewModal
- Retrofit, OkHttp
- Gradle

### Архитектурный паттерн: MVVM (Model-View-ViewModel)

### Компоненты системы

| Компонент | Описание |
|-----------|---------|
| **View** | Jetpack Compose интерфейсы для отображения чатов, профиля, контактов |
| **ViewModel** | Управление состоянием, обработка действий пользователя |
| **Repository** | Слой абстракции для работы с данными (локальные и удаленные) |
| **API Service** | Retrofit клиент для взаимодействия с REST API сервера |
| **Firebase** | Сервисы для хранения и передачи данных |
| **Backend Server** | Spring Boot приложение для обработки уведомлений |
| **Database** | Firebase Realtime Database для хранения пользовательских данных |
